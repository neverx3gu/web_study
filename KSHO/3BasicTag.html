모든 요소의 부모요소 -> 모든 요소는 html의 자식 요소 (!DOCTYPE 제외)

웹페이지에 단 하나만 존재

===========================

head Tag

메타데이터를 포함하기 위한 요소

웹 페이지에 단 하나만 존재

title / style / link / script / meta

title Tag

문서의 제목 정의

브라우저의 탭에 표시

style Tag

HTML문서를 위한 style 정보 정의

페이지 꾸미기

link Tag

외부 리소스와의 연계 정보 정의

주로 HTML과 외부 CSS파일 연계에 사용

script Tag

client-side Javascripit 정의

src 어트리뷰트로 외부 Javascripit 파일 로드 가능

meat Tag

description, keyword, author 등 메타 데이터 정의

브라우저, 검색엔진 등에 의해 사용
