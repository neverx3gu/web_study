제목 외에는 사용하지 않는 것을 추천

검색엔진이 이 제목 태그를 중요한 의미로 받아들일 수 있음

글자 형태 태그

=====================================

b

평범한 텍스트 p

볼드체 텍스트 b

p태그로 만든 볼드체

=====================================

strong

b태그와 동일하게 볼드체를 저장

b보다 의미적 중요성을 가짐

=====================================

i

이탈릭체를 지정

의미론적 중요성은 없다

=====================================

em

emphasized, 강조, 중요한

이탈릭인 i와 같지만 의미적 중요성을 가짐

=====================================

small

작은 글씨

그냥 작은 글씨

=====================================

mark

highlighted text 지정

=====================================

del

deleted text 지정

=====================================

ins

inserted text 지정

=====================================

sub / sup

아래로, 위로 쓰인 텍스트 지정

=====================================

본문 태그

=====================================

p

단락 지정

=====================================

br

개행 문자, 빈 요소이므로 종료 태그 없음

=====================================

pre

형식화된 text 지정

content가 작성된 그대로 브라우저에 출력
            var myArray = [];
            console.log(myArray.length); // 0

            myArray[1000] = true;  // [ , , ... , , true ]

            console.log(myArray.length); // 1001
            console.log(myArray[0]);     // undefined

hr

수평줄 삽입

빈 요소라 종료 태그 없음

q

짧은 인용문 지정

따옴표로 인용문 감싼다

blockquote

긴 인용문 지정

인용문을 들여쓰기하고, css로 다양한 style이 적용 가능함
